# Client Scripts

Client-owned systems live here.

- `Main.client.lua` — initializes client controllers
- `UI/HUDController.lua` — temporary HUD for round/data/hazard feedback
- `UI/BugSelectController.lua` — temporary bug selection and start-round UI
- `UI/RoundEndController.lua` — placeholder for future rewards screen
- `Camera/CameraController.lua` — placeholder for future tiny-bug camera tuning

The first UI is intentionally plain. It exists to test the game loop before final interface design.
