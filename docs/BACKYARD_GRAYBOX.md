# Backyard Graybox Notes

## Scale fantasy

The player is tiny. Ordinary backyard items should feel enormous.

## First arena layout

Start with one compact square/oval test map rather than a full open backyard.

Suggested zones:

1. **Bug Nest Spawn** — safe start area near a dirt mound.
2. **Grass Forest** — vertical blades of grass used for cover and visual scale.
3. **Patio Crumb Zone** — open, risky area with more crumbs.
4. **Flower Bed** — colorful landmark zone with petals and stems.
5. **Mulch Maze** — partial cover and simple navigation.
6. **Garden Tool Landmark** — shovel, glove, hose, or toy truck as a large set piece.

## MVP hazards

Use visual warnings before danger:

- Bird Shadow: a large moving shadow warns players to hide.
- Sprinkler Burst: lanes of water become dangerous.
- Shoe Stomp: warning circle appears before impact.

## Art rule

Make it readable before making it pretty. Players should instantly know:

- where they are
- where crumbs are
- what is dangerous
- where cover is
- how to return to the safe area
