# Interface Notes

## Temporary UI

The current interface is only for playtesting:

- left HUD: round state, player data, hazard warning
- right panel: choose bug and start round

This is not the final game UI.

## Final UI direction

The polished interface should feel like a playful bug lab / field notebook:

- chunky buttons
- leaf/paper panels
- DNA/crumb counters
- bug cards with strengths and weaknesses
- clear hazard warnings
- round-end reward reveal
