# Build a Bug MVP Plan

## North star

Build a small, playable Roblox experience where kids can see the full loop:

**Choose a bug → survive the backyard → collect crumbs → earn DNA → unlock cosmetics.**

## Milestone 1: Skeleton Playtest

Goal: prove the loop with placeholder graphics.

Deliverables:

- Rojo project syncs into Roblox Studio.
- Player can select Ant, Beetle, or Grasshopper.
- A round can start.
- Crumbs spawn in the arena.
- Touching crumbs awards crumbs/DNA.
- Hazard warnings appear on the HUD.
- Round end awards DNA.

## Milestone 2: Backyard Graybox

Build a simple test arena in Roblox Studio:

- Dirt floor
- Grass border
- 3–5 large cover objects
- A patio crumb zone
- A flower bed zone
- A mulch maze zone
- Spawn area / bug nest

Use simple parts first. Do not overbuild art before the loop is fun.

## Milestone 3: Bug Feel

Tune the first three archetypes:

- Ant: balanced, extra crumb value
- Beetle: slower, more health
- Grasshopper: faster, higher jump, less health

The goal is for kids to feel the difference quickly without needing a complex stat screen.

## Milestone 4: First Visual Pass

Replace graybox objects with stylized macro-backyard assets:

- Oversized grass blades
- Pebbles and dirt clumps
- Crumbs
- Flower stems/petals
- Garden hose
- Toy shovel
- Leaf cover

## Milestone 5: Builder Lite

Do not build full modular insects yet.

Start with safe cosmetic slots:

- Base bug
- Color
- Pattern
- Eyes
- Shell/back
- Trail/glow

## Kid contribution board

Good tasks for the kids:

- Name cosmetics
- Draw shell patterns
- Invent hazards
- Vote on bug abilities
- Playtest rounds
- Suggest badge names
- Help pick backyard objects

## Hard rule

No Battle Mode until Backyard Survival is fun for at least 10 minutes.
